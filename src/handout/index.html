#PrimaryVote! One-Page Quick Guide

Everything you need to know to compete — simplified.

Dates: Feb 13 – Mar 3, 2026 Deadline: Mar 3 • 5:00 PM Teams: 2 people required

1) Required Hashtags (No Exceptions)

  • #PrimaryVote!
  • #ChrisJonesforCongress

If either is missing → 0 points.

2) Polling Location Videos (30–60 sec, vertical)

Checklist items (claim only what you include):

  1. Address
  2. Where it is / cross streets
  3. Early voting timeline
  4. Primary + nonpartisan explanation
  5. Say “Vote for Chris Jones for Congress”

Points per checklist item:

  • Little Rock → 2 pts each
  • Major city limits → 3 pts each
  • All other areas → 4 pts each

3) Other Core Videos

  • Why I Support Chris Jones → 25 pts (max 2 videos)
  • Volunteer Event (Live) → 5 pts per event

4) Bonus Points

  • Cross-post bonus: +5 per additional platform
  • 8-County Sweep: +50 (one time)
  • Posting streak bonuses: +10 to +60
  • New team recruit: +20
  • Candidate interview: +5

5) How to Submit (No Confusion Policy)

  1. Post publicly
  2. Include required hashtags
  3. Tag the campaign
  4. Go to Team Portal → Submit a Video
  5. List expected points

If it’s not submitted, it doesn’t count.
