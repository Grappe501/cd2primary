AR-02 #PrimaryVote! Scavenger Hunt • Feb 13 – Mar 3, 2026 • Deadline 5:00 PM Mar 3

Teams: 2 people Format: Vertical video (30–60 sec) Submit: Team Profile

Required Hashtags (NO HASHTAGS = NO POINTS)

  • #PrimaryVote!
  • #ChrisJonesforCongress

How to Earn Points

1) Polling Location Explainer

Cover as many checklist items as possible:

  • Address
  • Where it is located
  • Early voting timeline (Feb 17 – Mar 3)
  • Primary + nonpartisan election
  • “Vote for Chris Jones for Congress”

Points per item vary by location category (Little Rock / major cities / all others).

2) “Why I Support Chris Jones”

  • 25 points each
  • Up to 2 per team

3) Campaign-Sanctioned Volunteer Event (Live)

  • 5 points per event
  • Must sign up in advance via Mobilize or campaign communication
  • Must post live while volunteering

Bonus Points

  • Cross-Post Platform Bonus: +5 for each additional platform you cross-post on (TikTok / Instagram / Facebook / X / Bluesky)
  • 8-County Sweep: +50
  • Posting Streaks: up to +60 (must claim)
  • Recruit a New Volunteer: +20 (celebratory video + signup)
  • Candidate Interview: +5
  • Distance Bonus: limited (non-major cities only)

Submit Everything (Required)

  1. Post the video
  2. Use required hashtags
  3. Tag the campaign
  4. Go to your Team Profile → Submit a Video
  5. Paste link(s) (one or more platforms) and list the points you expect (include cross-post bonus)

If you don’t list it, you don’t get it.
